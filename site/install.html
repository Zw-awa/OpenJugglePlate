Installation

安装

1. 克隆仓库

git clone https://github.com/Zw-awa/OpenJugglePlate.git
cd OpenJugglePlate

2. 当前建议的阅读顺序

  1. 先读根目录 README
  2. 根据兴趣选择 `hardware/`、`firmware/` 或 `software/`
  3. 准备协作时，再看 `CONTRIBUTING` 和 `SUPPORT`

3. 当前阶段说明

项目目前仍处于规划与原型开发阶段。此时最重要的是了解结构与方向， 而不是期待一个已经完整可编译、可装配、可运行的整机仓库。
