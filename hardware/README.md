# Hardware

[English](./README.md) | [简体中文](./README.zh-CN.md) | [繁體中文](./README.zh-TW.md)

This directory will hold the open hardware side of OpenJugglePlate:

- CAD
- printable parts
- fabrication notes
- BOM snapshots
- assembly instructions

The V1 baseline is a `3-actuator parallel platform` for a standard `40 mm` ping-pong ball.

Planned content:

- platform geometry
- actuator module design
- center constraint
- protection ring or guard
- mounting for camera, controllers, and power hardware
