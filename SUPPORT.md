## Where To Ask What

### Questions About The Project Direction

Open a `Question / Support` issue if you want clarification about:

- system architecture
- repository organization
- hardware/software responsibility boundaries
- contribution entry points

### Bug Reports

Open a `Bug Report` issue if:

- documentation is misleading or broken
- CI behavior is wrong
- a hardware/software behavior is reproducibly inconsistent with the docs

### Feature Ideas

Open a `Feature Request` issue if you want to propose:

- a new subsystem
- a documentation improvement
- a tooling or workflow enhancement
- a reproducibility improvement

## Before Opening An Issue

Please try to include:

- which area you are working on
- what you already read or tested
- what exact question, bug, or proposal you have
- any photos, logs, sketches, or links that help explain the context

## Scope Reminder

This repository is still in planning and prototyping.
Some questions may not have final answers yet, but the goal is to make the current assumptions explicit and discuss changes in the open.
